Fuji plot (Figure 1) GitHub

Fuji plot is a circos representation of multiple GWAS reseluts, which was designed to efficiently summarize significant SNPs across dozens of traits. We provide a wrapper script for Circos, generating input data and configuration files from GWAS data.

Genetic correlation corrplot (Figure 2) GitHub

We modified corrplot to visualize pairwise genetic correlations estimated via bivariate LD score regression.

Genetic correlation network (Figure 3) GitHub

Genetic correlation network was designed to visualize cross-phenotype relationships (pairwise genetic correlation matrix) across dozens of traits estimated via bivariate LD score regression.

Cell-type specificity network (Figure 6) GitHub

Cell-type specificity network was designed to summarize cell-type specific enrichments estimated via stratified LD score regression.

Installation

You can install any of my packages via the following code in R:

if (!require("devtools")) {
  install.packages("devtools")
}
devtools::install_github("mkanai/{package_name}")

Grimon (Kanai, M. et al., Bioinformatics, 2018)

GitHub

Kanai, M., Maeda, Y. & Okada, Y. Grimon: graphical interface to visualize multi-omics networks. Bioinformatics 34, 3934–3936 (2018). PubMed

Grimon (Graphical interface to visualize multi-omics networks) visualizes high-dimensional multi-layered data sets in three-dimensional parallel coordinates. It enables users to intuitively and interactively explore their data, helping their understanding of multiple inter-layer connections embedded in high-dimensional complex data.

ccmatch

GitHub

A package for optimal matching of cases to controls using network flow theory.

WRShd

GitHub

This package provides C++ sub-routines for the Harrell-Davis estimate functions from R. R. Wilcox’ robust statistics package WRS, which substantially speed up the original implementation in R.
